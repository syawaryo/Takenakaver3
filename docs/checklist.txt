項目 内容 チェック欄

■スリーブ図

=== 整合性 ===

#1 スリーブ位置決定に至った根拠図との整合を確認したか。
   ※根拠図も併せて提出する。シャフト・機械室は詳細図を作成すること。（根拠を確認できる程度）
   【必要図面】根拠図（設備ルート図等）
   【実装ランク】C（複数図面照合）→ Phase 3
   【備考】
   - 施工図＝直径やスリーブNoが書いてある図面（すぐわかる）
   - 根拠図＝スリーブ位置を決定した元の図面（設備ルート図等、要確認）
   - 施工図のスリーブ位置 vs 根拠図のスリーブ位置を座標マッチングで比較

#2 スリーブ用途・利用設備種別を記載したか。
   【実装ランク】B（VLM判定）→ Phase 2
   【手法】OCR + 設備種別キーワード検索、なければVLMで再確認
   【キーワード】給水/排水/空調ダクト/電気/衛生

#3 スリーブ呼び口径及び外径を記載したか。 例：200Φ（外径216）
   ※施工図のスリーブシンボルは外径の寸法とする。
   【実装ランク】B（VLM判定）→ Phase 2
   【手法】正規表現 /\d+Φ.*外径\d+/ + 呼び径→外径変換テーブル照合

#4 通り芯―芯の寸法と、スリーブ寄り寸法及びスリーブ間寸法の合計が合致することを確認したか。
   【実装ランク】A（OpenCV + OCR）→ Phase 2
   【手法】数値抽出 → 加算 → 照合（算術的に検証可能）

#5 勾配確保が必要なスリーブは、排水管のルート・流れ方向・勾配を確認し施工図に表記したか。（根拠図に着色し添付する。）
   【必要図面】排水系統図
   【実装ランク】C（複数図面照合）→ Phase 3
   【備考】
   - 「勾配確保が必要なスリーブ」＝自然流下（重力で流れる）排水管のスリーブ
   - 今回提供された図面には該当するものが見当たらない
   - 参考: https://www.shimada-hp.shizuoka.jp/fs/2/7/6/9/_/cafe_tenpohaitizu.pdf

#6 下階のスラブtoスラブの壁・複合耐火壁と干渉していないか。
   【必要図面】下階平面図 + 構造図
   【実装ランク】D（現時点では自動化困難）→ 手動
   【備考】
   - 構造図・調整図を見て比較する機能の実装は要検討
   - ベストはBIMで竹中JVがどんなフローでやっているかを知り、
     そのフローに一次チェックツールがハマるかを考えること
   - 3D的判断が必要

#7 段差スラブ近辺などスリーブ施工不可な場所となっていないか。（構造と調整済か。）
   【必要図面】同じ階の構造図（スラブ・梁伏図、段差、ドロップ）、同じ階の総合図（干渉調整図）
   【実装ランク】D（現時点では自動化困難）→ 手動
   【備考】
   - 「スリーブ施工不可」の具体例:
     - 段差スラブ（水回りの防水・排水エリア）
     - ドロップ（底仕上げ、設備用の段差）
   - 「構造と調整済か」＝施工不可場所にスリーブを設置している場合、
     構造側で補強や了承を得ているかの調整が入っているかどうか
   - 何がスリーブ施工不可にあたるのかは竹中JVに確認してもよい

=== 施工図表現 ===

#8 基準レベルと基準レベルからの寸法を記載したか。 例）1ＦＬ＋1750、2ＦＬ－550
   【実装ランク】B → Phase 1-2
   【手法】正規表現 /\d+FL[+\-]\d+/ パターンマッチ

#9 ある通り芯からのみの寄り寸法としていないか。（＝片側基準だけ問題）
   【実装ランク】A（OpenCV + OCR）→ Phase 2
   【手法】寸法線の参照先が複数通り芯かをOpenCVで検出

#10 型枠段差や凹み等からの寄り寸法としていないか。
    【実装ランク】B（VLM判定）→ Phase 2
    【備考】
    - スリーブ図に型枠段差が載っているかすら不明
    - 載っていたとしてもどれが型枠段差かの判別が困難
    - ルール: あくまでも通り芯からの寄り寸法にすべき
    - VLMに寸法線起点のクロップ画像を投入して判定

#11 あるスリーブ芯からの寄り寸法としていないか。
    【実装ランク】A（OpenCV）→ Phase 2
    【手法】寸法線の起点がスリーブ中心座標と一致するか幾何判定

#12 柱面や仕上げ面からの寄り寸法としていないか。
    【実装ランク】B（VLM判定）→ Phase 2
    【備考】
    - 仕上げ面は収まり調整で厚みが変わる可能性があるため基準にしない
    - あくまでも通り芯を基準にする
    - 柱面＝壁の外側線（二重線の外側）、柱の外周線（□の外側）
    - 躯体図がないと判定困難（スリーブ図単体には載っていない可能性）
    - 構造図・躯体図・施工図・調整図に載っている可能性あり

#13 建築、EPAスリーブの寄り寸法表記がバラバラでないか。
    【実装ランク】A（パターン比較）→ Phase 2
    【備考】
    - EPA＝設備系（電気E・配管P・空調A）をひとまとめにした略語
    - 建築側と設備側の図面を入力したときに寄り寸法表記の統一性を確認
    - 全スリーブの表記パターンを収集 → 外れ値検出

#14 スリーブNoは記載しているか。 例）SD‐001（協力会社頭文字‐番号）等
    【実装ランク】A（OCR + 正規表現）→ Phase 1
    【手法】正規表現 /[A-Z]{1,3}-\d{2,4}/ でマッチ
